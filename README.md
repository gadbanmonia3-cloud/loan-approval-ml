# Loan Approval Prediction

Machine Learning project for predicting loan approval decisions using classification models.

## Project Overview
This project includes:
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Model training and evaluation
- Final prediction generation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib
- Seaborn

## Files
- `loan_approval_prediction.ipynb` - Main notebook containing the full workflow
- `final_report.pdf` - Project report
- `predictions.csv` - Final prediction results

## Results
The best-performing model achieved strong classification performance using XGBoost.
